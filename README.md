# 🎬 YouTube PWA - Ad-Free with Background Play

A Progressive Web App (PWA) that enhances your YouTube experience with automatic ad-skipping and background audio playback. Watch videos without interruptions and listen to music even when your screen is off.

---

## ✨ Features

- 🚀 **Progressive Web App** - Install as a native-like app on any device
- ⏭️ **Automatic Ad Skipping** - Skip ads automatically, no more waiting
- 🔊 **Background Audio Playback** - Listen to music with screen off
- 📱 **Offline Support** - Watch cached videos without internet
- 🎵 **Media Controls** - Control playback from lock screen or notifications
- ⚡ **Fast Loading** - Service Worker caching for instant loads
- 🔒 **Privacy First** - No tracking, all processing happens locally
- 📡 **Works Offline** - Service Worker provides offline functionality

## 🚀 Quick Start

### Installation

#### Option 1: Mobile (Recommended)
1. Open this link on your mobile device
2. Tap the **"Install"** or **"Add to Home Screen"** button
3. Confirm installation
4. Launch the app from your home screen

#### Option 2: Desktop (Chrome/Edge)
1. Open this link in Chrome or Edge
2. Click the **Install** button (top-right corner)
3. Confirm installation
4. The app will open in a new window

#### Option 3: Manual
1. Go to `Settings` → `More Tools` → `Create Shortcut` (Chrome)
2. Check "Open as window"
3. Click Create

## 📋 How It Works

### 🎯 Automatic Ad Skipping
- Monitors YouTube for skip buttons
- Automatically clicks when available
- Removes ad overlays and countdown timers
- Works with most YouTube ad types

### 🔊 Background Audio
- Continues playing audio when app is minimized
- Updates lock screen controls
- Supports media key controls
- Prevents screen lock while playing

### 💾 Offline Support
- Service Worker caches essential files
- Stores recently watched videos
- Works without internet connection
- Auto-syncs when connection returns
